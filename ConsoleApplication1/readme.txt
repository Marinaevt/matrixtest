Описание dll библиотеки ImprovedSystem версии 1 от 4 октября 2006 г.

Библиотека содержит функции:

Функция - ImprovedGaussSystem
Описание: предназначена для решения СЛАУ методом Гаусса
Примечание: 
1. Числа в матрицах должны быть Double - 8 байт 
2. В этой версии она не поддерживает многопоточные вычисления
Параметры: 
function ImprovedGaussSystem(A,C,B,PB:Pointer;NN,Q:Integer):integer;cdecl;
Входные:
A - массив матрицы A из чисел Double(8 байт)(левая часть СЛАУ) ,массив должен быть длинны NN*Q*8 байт и содержать
ленточную форму(верхнюю половину ленты(симметричную нижней)) л.ч. матрицы по строкам.
C - массив матрицы С из чисел Double(8 байт)(правая часть СЛАУ) ,массив должен быть длинны NN*8 байт и содержать вектор правой части СЛАУ
B - массив должен быть длинны Q*8 байт, содержание не имеет значения
PB - массив должен быть длинны Q байт, содержание не имеет значения
NN - число строк матрицы A
Q - число столбцов(ширина полосы(половина ширины ленты)) матрицы А
Результат: 
функция возвращает номер свой версии, в данной версии это всегда 1
В массиве C находится решение СЛАУ
В массивах A,B и PB находятся неизвестные числа (массивы содержат мусор)
